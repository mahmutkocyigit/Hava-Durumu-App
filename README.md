# 🌤 Modern Hava Durumu Uygulaması (Glassmorphism)

Bu proje, **OpenWeatherMap API** kullanılarak geliştirilmiş, şık ve kullanıcı dostu bir hava durumu uygulamasıdır. Kullanıcıların istedikleri şehri aratarak anlık hava durumu bilgilerine ve 5 günlük hava tahminlerine ulaşmasını sağlar.

## ✨ Özellikler

* **Anlık Veri:** Şehir bazlı sıcaklık, hissedilen sıcaklık, nem, rüzgar hızı ve basınç bilgileri.
* **5 Günlük Tahmin:** Gelecek 5 güne ait hava durumu öngörüleri.
* **Dinamik Arka Plan:** Hava durumuna göre (Güneşli, Bulutlu, Karlı, Yağmurlu vb.) otomatik değişen gradyan arka planlar.
* **Glassmorphism Tasarım:** Modern, şeffaf ve estetik "cam efekti" arayüzü.
* **Tam Duyarlı (Responsive):** Mobil, tablet ve masaüstü cihazlarla tam uyumlu tasarım.
* **Görsel Detaylar:** Hareketli arka plan parçacıkları ve hava durumuna uygun ikonlar.

## 🛠 Kullanılan Teknolojiler

* **HTML5:** Semantik yapılandırma.
* **CSS3:** Custom Variables, Flexbox, Grid, Animations ve Glassmorphism efektleri.
* **JavaScript (ES6+):** Fetch API (Asenkron veri çekme), DOM manipülasyonu ve dinamik filtreleme.
* **OpenWeatherMap API:** Gerçek zamanlı hava durumu verileri için.
* **Google Fonts:** Projede şıklık için 'Outfit' fontu kullanılmıştır.

## 🚀 Kurulum ve Çalıştırma

Projeyi yerel bilgisayarınızda çalıştırmak için şu adımları izleyebilirsiniz:

1.  Bu depoyu klonlayın:
    ```bash
    git clone [https://github.com/mahmutkocyigit/Hava-Durumu-App.git](https://github.com/mahmutkocyigit/Hava-Durumu-App.git)
    ```
2.  Proje klasörüne gidin:
    ```bash
    cd Hava-Durumu-App
    ```
3.  `index.html` dosyasını tarayıcınızda açın.

> **Not:** Uygulamanın çalışması için geçerli bir OpenWeatherMap API anahtarına ihtiyacınız vardır. Mevcut anahtar `app.js` dosyası içerisinde tanımlıdır.
